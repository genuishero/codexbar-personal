# CodexBar Windows Version Resources

This directory contains resources for the Windows version of CodexBar.

## Icon

The `icon.png` file is a placeholder icon. For Windows builds, you need to convert it to `icon.ico` format.

You can use online converters or tools like:
- ImageMagick: `convert icon.png icon.ico`
- Online: https://convertio.co/png-ico/

The icon should be 256x256 pixels for best quality.
